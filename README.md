# Pharma-Tools

Application web destinée à simplifier la compréhension des interactions médicamenteuses.

## Architecture

* **Frontend :** Astro, React, Nano Stores.
* **Backend :** FastAPI, Python, LangChain.

## Installation

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn api.main:app --reload
```
